# Jay Ar & Princess Wedding Website (Beige, Light, Animated)

### Why minsan “wala effects”?
- **You must run via localhost** (Live Server / python http.server), not by double-clicking `index.html`.
- **Need internet** for GSAP/Lenis CDN scripts. (Petals + countdown still work kahit walang GSAP.)

## Run locally (Windows)
1) Extract ZIP
2) Open terminal inside the folder:
```bash
python -m http.server 5173
```
3) Open: http://localhost:5173

## Files
- `index.html`
- `assets/main.css`
- `assets/main.js`
- `assets/hero.jpg` (your photo)
- `assets/wedding-ambient.wav` (sample music)
