# Jay Ar & Princess Wedding Website (Static)

This is a static wedding website (HTML/CSS/JS) with:
- GSAP animations + reveal on scroll
- Lenis smooth scrolling
- Unique countdown (flip animation + orbit ring)
- RSVP modal (demo: saves locally + JSON download)
- Gallery lightbox

## Run locally
Just open `index.html` in your browser.

## Deploy to GitHub + Vercel
1) Create a new GitHub repo
2) Upload all files in this folder (keep structure)
3) In Vercel: **New Project** → import the GitHub repo → Framework: **Other** → Build command: *(none)* → Output: *(default)*
4) Deploy

## Customize
Open `index.html` and edit:
- Names, venue details, schedules, FAQ text
In `assets/main.js`:
- `WEDDING_DATE` (Asia/Manila)

Enjoy ✨
